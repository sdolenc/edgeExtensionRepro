Adding Files for Your Course

To add files to use in your course, click Upload New File. Then follow the prompts to upload a file from your computer.

Caution: Microsoft Learning recommends that you limit the file size to 10 MB. In addition, do not upload video or audio files. You should use a third party service to host multimedia files.

The course image, textbook chapters, and files that appear on your Course Handouts sidebar also appear in this list.

Using File URLs

Use the Studio URL value to link to the file or image from a component, a course update, or a course handout.

Use the Web URL value to reference the file or image only from outside of your course. Note: If you lock a file, the Web URL no longer works for external access to a file.

To copy a URL, double click the value in the URL column, then copy the selected text.
